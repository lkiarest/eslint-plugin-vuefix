# eslint-plugin-vuefix

根据eslint配置的规则自动修复(autofix) .vue 文件

## Installation

```
$ npm install eslint-plugin-vuefix --save-dev
```

__需要与eslint使用同样的方式安装__

## Usage

在“.eslintrc” 文件中添加插件声明

```json
{
    "plugins": [
        "vuefix"
    ]
}
```
